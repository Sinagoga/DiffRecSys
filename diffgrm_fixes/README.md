# Исправления для репозитория DiffGRM

Эта папка содержит исправленные версии файлов из официального репозитория
**DiffGRM** (`https://github.com/liuzhao09/DiffGRM`). Каждое исправление проверено
по реальному коду (а не по описаниям) и прогнано через `py_compile`.

Файлы здесь лежат в той же структуре путей, что и в оригинальном репозитории —
просто замените одноимённые файлы в клоне DiffGRM на эти.

---

## Как применить

### Вариант A — заменить файлы (просто)
Скопируйте файлы из этой папки в ваш клон DiffGRM, сохраняя пути:

```bash
# из корня вашего клона DiffGRM:
cp <путь>/diffgrm_fixes/main.py                                  ./main.py
cp <путь>/diffgrm_fixes/requirements.txt                         ./requirements.txt
cp <путь>/diffgrm_fixes/genrec/models/DIFF_GRM/model.py          ./genrec/models/DIFF_GRM/model.py
cp <путь>/diffgrm_fixes/genrec/models/DIFF_GRM/trainer.py        ./genrec/models/DIFF_GRM/trainer.py
cp <путь>/diffgrm_fixes/genrec/models/DIFF_GRM/tokenizer.py      ./genrec/models/DIFF_GRM/tokenizer.py
cp <путь>/diffgrm_fixes/genrec/models/DIFF_GRM/evaluator.py      ./genrec/models/DIFF_GRM/evaluator.py
# опционально (только если запускаете baseline AR_GRM):
cp <путь>/diffgrm_fixes/genrec/models/AR_GRM/model.py            ./genrec/models/AR_GRM/model.py
```

### Вариант B — применить patch (если файлы репозитория уже менялись)
В папке лежит `all_fixes.patch` (вывод `git diff`). Из корня клона DiffGRM:

```bash
git apply --check <путь>/diffgrm_fixes/all_fixes.patch   # проверка
git apply         <путь>/diffgrm_fixes/all_fixes.patch   # применение
```

### Подробнее о ключевых исправлениях

**1. `model.py` — захардкоженная «4» (самый важный фикс для масштабирования).**
В оригинале guided-маскирование жёстко ограничено 4 шагами. При стандартном
конфиге DiffGRM (`n_digit=4`) это безвредно — `min(g, 4, 4) == min(g, 4)`,
то есть фикс **не меняет воспроизведение статьи**. Но если вы поднимаете
`n_digit` (например до 10, как в проекте DiffRecSys), оригинал молча обучает
только 4 из 10 позиций. Фикс делает число шагов равным `n_digit`. Проверено:
shape-проверки в `forward()` (`B_expanded = B * augment_factor`) остаются
согласованными, т.к. `augment_factor` берётся из того же `guided_steps`.

**2. `trainer.py` — краш при загрузке чекпойнта.**
Чекпойнт пишется только внутри блока оценки при улучшении метрики. Если
обучение короче `eval_start_epoch` (по умолчанию 20) или ни один eval не
улучшил результат — файла нет, и в самом конце `fit()` падает `FileNotFoundError`,
теряя весь прогон. `os` уже импортирован в начале файла, поэтому guard безопасен.

**3. `AR_GRM/model.py` — опциональный фикс baseline-модели.**
Это **не** DiffGRM, а авторегрессионный baseline для сравнения. Под штатным
симметричным конфигом beam'ов (`pre_cut = beam = 128`) краша нет, но токен всё
равно может приписываться не к тому родительскому лучу. Применяйте, только если
будете запускать AR_GRM (особенно с асимметричным beam-конфигом).

## Что мы НАМЕРЕННО НЕ трогали (и почему)

- **`config.yaml`** — не меняли. Штатный конфиг (`n_digit=4, codebook_size=256,
  sent_emb_pca=256`) корректен для воспроизведения статьи. Менять его на
  `n_digit=10 / 512` — это **адаптация под наш проект**, а не исправление бага.
- **Ограничение OPQ: `sent_emb_pca % n_digit == 0`** — это **не баг**, а
  требование FAISS. При штатном `256 % 4 == 0` всё работает. Если будете
  масштабировать до `n_digit=10`, поставьте `sent_emb_pca` кратным 10 (напр. 250)
  и держите `opq_use_gpu: False` (GPU-PQ в FAISS поддерживает только 8 бит,
  а `codebook_size=512` → 9 бит). Это настройка, не правка кода.
- **Коллизии SID** (`tokens2item` перезаписывает совпадающие SID),
  **OCN/guided-objective смещает ELBO**, **CPD = greedy MAP**, **отсутствие 1/t
  взвешивания в loss** — это **проектные решения / спорные места**, а не
  однозначные баги. Мы их не трогали (как вы и просили: «если это не баг — оставь»).
- **«Дебанкнутые» претензии** — коллизия MASK с кодом 0, «ненадёжный» KV-cache,
  off-by-one в beam-цикле, +1 в индексации item_id — проверены повторно: всё это
  **корректный код**, не баги. Не трогали.
